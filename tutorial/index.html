从安装、登录到第一次任务，按你的入口选择路径

第一版覆盖桌面 App、CLI、IDE 扩展和 Web。Windows 桌面 App 路径最详细，适合新手照着走。

01

Windows 桌面 App

  • 1从官方入口下载并安装 Codex app，或使用 Microsoft Store/winget 路径。
  • 2打开 Codex，使用 ChatGPT 账号或 OpenAI API key 登录。
  • 3选择要处理的项目文件夹，确认使用 Local 在本机工作。
  • 4发送第一条具体需求，例如“帮我检查这个页面为什么报错”。
02

CLI

适合熟悉终端的人。Codex CLI 可以在本地目录读取、修改并运行代码。

03

IDE 扩展

适合在 VS Code 等编辑器旁边使用 Codex，让它结合当前代码上下文工作。

04

Web

适合把 GitHub 仓库任务交给云端 Codex，生成分支或 PR 结果。
